## Decisions Made

1. **Removed `thinking_level` from math verifier** — Conflicts with `response_mime_type="application/json"`. Gemini mixes thinking tokens into JSON output, corrupting parsing.
2. **Tier 2 always fires in Red Zone** — Sequential flow, not parallel. Tier 1 (0.05ms) + Tier 2 (~5s) total ~5.6s. `asyncio.gather` overhead not worth it for 0.05ms Tier 1.
3. **Bug fix: unescaped `{}` in TIER2_VERIFICATION_PROMPT** — Python `.format()` interpreted `{ "verified": ... }` as format variable → `KeyError`. Tier 2 verification was **silently broken** in all prior production deployments.
