## Decisions Made

1. **`.find()` over array index** — Citation resolution now uses `sources.find(s => s.id === N || s.article_number === 'N')` instead of `sources[N-1]`. Resilient to non-sequential IDs.

2. **ZWS injection (Option C)** — Zero-Width Space (`\u200B`) injected into citation brackets `[N]` → `[\u200BN]` before ReactMarkdown processing. Prevents ReactMarkdown from parsing `[N]` as link references. CitationText regex updated with `\u200B?` to match both formats.

3. **Footer preserved** — Sources footer (`📚 წყაროები:`) is only stripped during streaming (not after completion). ZWS makes it safe for ReactMarkdown.

4. **`SourceDetail.id: number`** — Frontend type aligned with backend's sequential integer IDs.
